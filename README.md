## Getting Started

## Installation

1. Clone the repo
   ```sh
   git clone git@github.com:shavinski/react-jobly.git
   ```
2. Install dependencies
   ```sh
   npm install 
   ```
3. Start the local server
   ```sh
   npm start
   ```

Link to back-end code: 
https://github.com/shavinski/express-jobly
